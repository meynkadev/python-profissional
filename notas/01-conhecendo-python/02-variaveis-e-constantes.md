# Conhecendo Python

## Variáveis e Constantes

### Variável

Uma variável é um nome que referencia uma informação armazenada na memória.

Exemplos:

numero = 1 (a variável numero aponta para a informação de valor 1, que está na memória)
nome = "Meynkâ" (a variável nome aponta para a informação de valor "Meynkâ", que está na memória)

Os valrores das variáveis podem mudar no decorrer da ultilização do programa. 

### Constante

Python não possui constantes verdadeiras. 

Porém, o coneceito de constantes se referem a um valor que não deve ser alterado. 

A comunidade utiliza letras MAIÚSCULAS para indicar que aquele valor não deve ser alterado. 

Exemplos: 

PI = 3.14259
URL_API = "https://api.exemplo.com
