# Conhecendo Python

## Tipos de dados

- str -> texto
- int -> inteiro
- float -> decimal
- bool -> verdadeiro ou falso

## Funções úteis

### type()

Mostra o tipo da variável.

### dir()

Mostra tudo que um objeto sabe fazer.

### help()

Mostra a documentação de uma função ou método.

Exemplo:

help(str.count)
help(str.replace)
help(print)
